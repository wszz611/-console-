# Switch Console 交换机无线调试工具
## 项目简介
基于STM32F103C8T6开发的便携式锐捷交换机Console调试设备，完成硬件原理图、PCB设计与嵌入式固件全流程开发，支持蓝牙无线连接手机发送调试指令，替代传统串口线调试交换机，大幅提升现场设备调试效率，完整工程开源分享。

## 核心技术栈
- 编程语言：标准C语言（STM32 HAL库开发）
- 硬件外设：UART串口、蓝牙模块、GPIO按键、（64×128像素 0.96—1.54均可）OLED显示、rj45座子
- 硬件工具：立创EDA（原理图+双层PCB Layout）
- 调试工具：逻辑分析仪、串口示波器

## 功能亮点
1. 有限状态机实现多级菜单交互，内置交换机常用调试指令快捷发送
2. 蓝牙BLE无线透传，手机小程序/串口助手远程下发配置命令
3. 完整硬件工程开源，包含原理图zip、全部源码
4. 解决传统有线Console线束缚问题，现场调试效率提升40%
5. 完善异常容错处理，修复多组串口时序、数据丢包驱动bug

## 仓库目录说明
├─ switch_tool.zip # STM32 固件完整工程（Keil）
└─ 原理图.zip # 立创 EDA 硬件原理图

## 编译与使用步骤
1. 下载 `switch_tool.zip` 解压，烧录进入stm32
2. 硬件：烧录固件后上电，蓝牙名称 `Switch_Debug` 手机连接
3. 打开串口助手，波特率9600即可和交换机Console口通信

## 注意!!!!!!
  交换机console口的线序与平时rj45网线的线序不同 连接时如果不通注意排查线序问题
